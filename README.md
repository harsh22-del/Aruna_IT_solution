# Nexaforge — Industrial Engineering Website

A premium, futuristic, fully-static multi-page website for an industrial engineering and technical services company. Pure HTML / CSS / vanilla JavaScript — no build step.

## Run locally

1. Open the project folder in **VS Code**.
2. Install the **Live Server** extension (Ritwick Dey).
3. Right-click `index.html` → **Open with Live Server**.

Or simply double-click `index.html` to open in a browser.

## Pages

- `index.html` — Home (hero, services, stats, why-us, projects, testimonials, CTA)
- `about.html` — Company story, mission, timeline, team
- `services.html` — 12 services + delivery methodology
- `projects.html` — Project showcase grid + impact stats
- `certifications.html` — Accreditations + achievement timeline
- `contact.html` — Contact form, info cards, FAQ accordion
